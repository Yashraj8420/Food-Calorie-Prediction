# Food-Calorie-Prediction
A Machine Learning web application developed using Python and Streamlit that predicts the calorie content of various food items based on the selected food and quantity. The application also displays nutritional information such as Protein, Carbohydrates, and Fat, helping users make healthier dietary choices.

🚀 Features
🍴 Predict calories for different food items
⚖️ Quantity-based calorie calculation
💪 Displays Protein, Carbohydrates, and Fat
🎨 Modern and responsive Streamlit interface
🤖 Machine Learning model using Scikit-learn
📊 Easy-to-use interactive dashboard
